# 🌍 AI Language Translation Tool

An AI-powered Language Translation Tool built using Python and Streamlit.  
This application allows users to translate text between multiple languages instantly with text-to-speech support.

## 🚀 Features

- 🌐 Multi-language Translation
- 🔊 Text-to-Speech Audio
- 📋 Copy Translation
- ⬇ Download Audio
- 🎨 Professional User Interface

## 🛠 Technologies Used

- Python
- Streamlit
- Deep Translator
- gTTS

## 📦 Installation

```bash
pip install -r requirements.txt
```

## ▶ Run the Application

```bash
streamlit run app.py
```

## 📸 Project Output

The application translates text instantly and provides audio playback for translated text.
